# 贡献指南

感谢你对本项目的关注。欢迎通过 Issue 反馈问题、通过 Pull Request 贡献代码或文档。

## 如何贡献

### 报告问题

- 在本仓库的 **Issues** 中新建 Issue。
- 请尽量写清：环境（Node 版本、系统）、复现步骤、期望行为与实际行为、相关日志或截图。

### 提交代码

1. **Fork** 本仓库，并 clone 到本地。
2. 基于 **main** 新建分支（例如 `fix/xxx`、`feat/xxx`）。
3. 在本地完成修改，确保：
   - `npm run lint` 通过；
   - `npm run build` 能成功（若涉及构建）。
4. 提交信息尽量简洁明确（如 `fix: 修复天气工具中文城市无结果`、`feat: 支持代码块高亮`）。
5. 推送到你 Fork 的仓库，然后打开 **Pull Request** 指向本仓库的 **main**。
6. 在 PR 中简要说明改动目的与方式，维护者会尽快 review。

### 行为准则

- 尊重他人，就事论事，不进行人身攻击。
- 讨论技术时保持友好、建设性。

再次感谢你的参与。
