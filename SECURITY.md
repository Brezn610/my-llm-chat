# 安全政策

## 支持的版本

当前我们会对 **最新主分支** 上的代码进行安全相关更新。

## 如何报告漏洞

若你发现与安全相关的漏洞（例如可能导致未授权访问、数据泄露、注入等），请**不要**在公开的 Issue 中披露。

请通过以下方式私下报告：

- **邮箱**：在仓库 About 或作者主页中提供的联系邮箱；或  
- **GitHub Security Advisories**：在仓库的 **Security** 标签页选择 “Report a vulnerability”，按指引提交。

我们会尽快确认并回复，并在修复后（若适用）在发布说明或 Security 中致谢（除非你希望匿名）。

感谢你负责任地披露问题。
